# 3 图片相关的优化

##1.一张图片的解析过程

#### jgp:

1. 颜色空间的转换，从RGB转换成其他
2. 重采样，区分高频和低频的颜色变换
3. DCT：对高频颜色进行压缩

#### png8/png24/png32的区别

- png8：256色+支持透明
- png24：2^24色+不支持透明
- png32：2^24色+支持透明

## 2.优化方式：

1. 精灵图：减少http请求数量
2. 图片压缩
3. 将需要加载的图片设置为背景图；
4. 将图片设置为base64；减少请求数量
5. 使用svg矢量图，使用iconfont解决icon问题
6. 在安卓下使用webp，IOS不兼容
